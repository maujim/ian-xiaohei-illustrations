# Prompt Examples

The following prompts can be copied directly into Codex.

## Illustration Planning Only

```text
Use $ian-xiaohei-illustrations. Do not generate images yet.
Please analyze where the article below would benefit from illustrations, and output a shot list of about 5 images.
For each image, clearly state:
- Which paragraph it should follow
- The image subject
- The core idea
- The structural type
- What Xiaohei is doing in the image
- Suggested elements
- Suggested English annotation words

<paste article>
```

## Article Body Illustrations

```text
Use $ian-xiaohei-illustrations to generate 4 grotesque Xiaohei body illustrations for the article below.
Requirements: 16:9 landscape, pure white background, black hand-drawn line art, and a few red, orange, and blue handwritten English annotations.
Each image should express only one core structure. Do not make PPT-style infographics or cute cartoons.

<paste article>
```

## Long-Form Illustration Strategy

```text
Use $ian-xiaohei-illustrations to create an illustration strategy for this long-form article.
Do not distribute illustrations evenly. Choose only cognitive anchors: the core judgment, input-output loops, before-and-after contrasts, common pitfalls, and handoff paths.
Default to 6-8 images. Output the shot list first; do not generate images.

<paste article>
```

## Generate One Image for a Single Point

```text
Use $ian-xiaohei-illustrations to generate one 16:9 body illustration for this point:

Trust is not declared out loud; it is laid down piece by piece with evidence.

The image should be grotesque but clean, and Xiaohei must perform the core action.
Use at most 5 short English annotations.
```

## Workflow Theme

```text
Use $ian-xiaohei-illustrations to generate an image for "turning one piece of raw material into three kinds of content: traffic, trust, and conversion."
Do not draw a formal flowchart, and do not recreate the old "one fish, many dishes" example.
Please invent a new low-tech metaphor and have Xiaohei participate in the core action.
```

## Image Edit: Remove the Title

```text
Use $ian-xiaohei-illustrations to help me edit this image.
Remove the "Workflow / Flowchart" title and underline in the upper-left corner, while keeping everything else unchanged.
Do not add any new text or objects.
```

## Image Edit: Make Xiaohei More Involved

```text
Use $ian-xiaohei-illustrations. The direction of this image is right, but Xiaohei feels a bit decorative.
Please keep the core idea unchanged and regenerate a version where Xiaohei is the person truly driving the structure.
Make the image a little more grotesque, while keeping it pure white, clean, and light on text.
```

## Generate a Set of Style Samples

```text
Use $ian-xiaohei-illustrations to output 5 Xiaohei body illustration samples on different themes.
The themes should cover: information overload, product validation, content compounding, one-person company, and trust building.
Generate each image separately; do not combine them into one image.
```
